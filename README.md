# Portfolio Website

A modern, responsive portfolio website featuring a cosmic-inspired design with animated star backgrounds, interactive project carousels, and immersive user experiences.

## ✨ Features

### 🌌 Visual Experience
- **Cosmic Background**: Multi-layered animated star field with randomized patterns
- **Organic Design Language**: Garden and cultivation metaphors throughout
- **Immersive Layout**: Background animations visible through transparent sections
- **Purple Cosmic Theme**: Professional space-inspired color palette

### 🎠 Interactive Project Showcase
- **Multi-Image Carousels**: Browse through multiple screenshots per project
- **Full-Screen Image Popup**: Click any image for detailed viewing
- **Looping Navigation**: Seamless browsing with arrow keys and touch gestures
- **Auto-Play Carousels**: Smooth transitions with pause-on-hover
- **Responsive Controls**: Touch-friendly navigation on all devices

### 🛠️ Modern UX
- **Smooth Animations**: Parallax scrolling and organic transitions
- **Accessibility**: Full keyboard navigation and screen reader support
- **Mobile-First Design**: Optimized for all screen sizes
- **Performance**: CSS-only animations for smooth 60fps experience

### 🏢 **WorkTree**
Airbnb-style platform for co-working space bookings with clean interface design.
- **Tech**: Ruby on Rails, PostgreSQL, Devise

### 🎨 **PsiSketch**
Mobile-first remote viewing practice app with guided exercises and session tracking.

- **Images**: 6 screenshots including interface and session environments

### 🤖 **Muse Assistant**
Chrome extension enhancing Zendesk workflows with AI-powered automation.

### 🎬 **FilmPicker**
CLI movie discovery tool with genre-based filtering and favorites.

- **Images**: 4 screenshots of menu, selection, and results

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Grid, Flexbox, Custom Properties
- **Animations**: CSS keyframes, transforms, transitions
- **Images**: Local asset management with lazy loading
- **Fonts**: Inter font family from Google Fonts
- **Icons**: Font Awesome for UI elements

## 🎨 Animation System

### Star Field Background
- **Multi-Layer Design**: 3 independent animated layers
- **Randomized Patterns**: Prime number dimensions (397×331px, 367×307px)
- **Organic Movement**: Different speeds and twinkling rates
- **Shooting Stars**: Periodic diagonal light trails
- **Purple Nebulae**: Subtle cloud-like gradients

### Interactive Elements
- **Carousel Navigation**: Smooth image transitions with dots and arrows
- **Modal Popups**: Full-screen image viewing with navigation
- **Hover Effects**: Subtle transformations and glows
- **Scroll Animations**: Staggered reveals and parallax movement

## 📱 Responsive Design

- **Desktop**: 1200px+ (Full carousel controls, large star field)
- **Tablet**: 768px-1199px (Adapted controls, optimized spacing)  
- **Mobile**: Below 768px (Touch gestures, compact layout)
- **Small Mobile**: Below 480px (Streamlined interface)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash

   ```

2. **Open in browser**
   ```bash
   open index.html
   ```

3. **No build process required** - Pure HTML/CSS/JS!
